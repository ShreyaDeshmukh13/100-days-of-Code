### Mini Project
This project is a simple **Movie Ranking webpage** created using HTML.

It displays a list of my top 3 favorite movies, along with short descriptions.  
The page demonstrates how to structure content using headings and paragraphs in a clean and readable format.

### Features
- Clear hierarchical structure using headings
- Simple and readable layout
- Organized presentation of movie preferences
