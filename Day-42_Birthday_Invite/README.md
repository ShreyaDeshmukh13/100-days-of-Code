# 🎂 Birthday Invite Webpage

## 📌 Project Overview

This project is a simple **Birthday Invitation Webpage** created using basic HTML.
It displays an invitation with event details, an image, and a location link.

---

## 🎯 Objective

The goal of this project is to practice:

* HTML structure
* Using headings, images, lists, and links
* Creating a basic webpage layout

---

## 🛠️ Technologies Used

* HTML5

---

## 📂 Features

* Displays birthday event details
* Includes a birthday cake image
* Lists items to bring
* Provides a Google Maps link for location

---

## 📸 Preview

A simple webpage showing:

* Title: *It's My Birthday!*
* Date of event
* Image of a birthday cake
* List of items to bring
* Location link

---

## 🚀 How to Run

1. Download or clone the repository
2. Open the `index.html` file in any browser
